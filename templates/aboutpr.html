This project is a movie site that aims to provide an immersive experience for movie enthusiasts by offering them a range of functionalities. The site features a navigation bar that allows users to access various pages, including Login page,Signup Page and Contact page which help user getting in touch with me.All the pages are made using HTML,CSS, Bootstrap, Tailwind CSS and Javascript(used to add functionality in password bars and links for movie cards on index page). The Login and Signup pages collect user information and store it in the backend using flask and mysql database. The site's home page includes a recommend button that provides movie recommendations to users based on their inputs.The sentiment analysis is done for the reviews and based on them it tells us whether they are positive or negative.This recommendation system is powered by a machine learning algorithm that ensures personalized and accurate movie suggestions.
